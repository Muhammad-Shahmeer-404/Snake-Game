#A Windows Terminal Based Snake Game

A Classic snake game witten in C for the windows terminal. The game features multiplayer and a single player mode additionally it has a Highscore Tracker.

Features
* **Single Player:** Classic snake gameplay.
* **Multiplayer:** Play against a friend locally.
* **High Scores:** Tracks your best performance.
* **Smooth Controls:** Optimized for Windows terminal input.

* How to Run
Since this is a C project, you need to compile it first.

1.  **Clone the repository:**
    ```bash
    git clone git@github.com:YourUsername/snake-game-c.git
    cd snake-game-c
    ```

2.  **Compile the game:**
    ```bash
    gcc main.c Singleplayer.c Multiplayer.c -o snake_game
    ```

3.  **Run it:**
    ```bash
    ./snake_game
    ```
## Controls
**Sigleplayer Mode** WASD
**Multiplayer Mode** player_1 --> WASD player_2 ---> Arrow Keys
